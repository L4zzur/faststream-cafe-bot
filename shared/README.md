# Shared Module

Общая библиотека для всех микросервисов проекта **FastStream Cafe**.

Этот модуль содержит переиспользуемый код, общий для `Order`, `Inventory` и `Bot` сервисов, что позволяет избежать дублирования и поддерживать согласованный формат данных.

## Структура

- **`schemas.py`**: Pydantic-модели данных.
  - Определяет формат сообщений, передаваемых через RabbitMQ (например, `OrderIn`, `Product`, статус заказа).
- **`topics.py`**: Константы имен очередей и топиков RabbitMQ.
  - Гарантирует, что все сервисы подписываются и отправляют сообщения в одни и те же каналы.
- **`logger.py`**: Единая конфигурация логирования.
  - Обеспечивает единообразный формат логов во всех сервисах.
- **`config.py`**: Базовые настройки конфигурации (например, настройки подключения к RabbitMQ).
  - Использует `pydantic-settings` для валидации переменных окружения.

## Использование

Данный модуль устанавливается в локальном окружении как editable-пакет через `uv` workspace. В `pyproject.toml` на корневом уровне он определен как член workspace.
